This is a [SvelteKit](https://svelte.dev/docs/kit/introduction) app inside the `oxtro-ui` Turborepo workspace.

## Development

Run the app from the workspace root:

```bash
bun dev --filter=docs
```

Or from this app directory:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

Edit `src/routes/+page.svelte` to update the landing page.

## Useful scripts

```bash
bun check
bun build
bun preview
```
